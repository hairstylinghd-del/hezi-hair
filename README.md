# ✂️ HeziCut — מערכת ניהול מספרה

מערכת ניהול תורים מלאה עם סנכרון real-time ותזכורות WhatsApp.

## תכונות
- ניהול תורים, לקוחות ושירותים
- סנכרון real-time בין כל המכשירים
- תזכורות WhatsApp אוטומטיות (יום לפני + שעה לפני)
- PWA — עובד על אייפון כמו אפליקציה

## טכנולוגיות
- Next.js 14
- Supabase (PostgreSQL + Realtime)
- Green API (WhatsApp)
- Vercel (Hosting)
